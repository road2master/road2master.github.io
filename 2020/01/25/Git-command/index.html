git 修改分支名称

本地分支重命名

git branch -m oldName newName

远程分支重命名

  • 重命名（对应远程分支）的本地分支

    git branch -m oldName newName

  • 删除远程分支

    git push --delete origin oldName

  • 上传新命名的本地分支

    git push origin newName

  • 把修改的本地分支与远程分支关联

    git branch --set-upstream-to origin/newName

查看当前代码仓库源

  • 查看当前源

    git remote -v

  • 重设

    git remote set-url origin xxxx_url
