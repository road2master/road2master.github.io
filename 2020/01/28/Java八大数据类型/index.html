Java八大数据类型

发表于2020-01-28|更新于2022-02-22|Java
|字数总计:119|阅读时长:1分钟|阅读量:

字符类型　　　char　　　　16位 存储Unicode码，用单引号赋值

布尔类型　　　boolean 　　true 　false

数值类型

　　1.整数类型 byte 　 8位（-128~127）

　　　　　　　short 　16位（-32768~32767）

　　　　　　　int 　　32位（负2的31次方~2的31次方-1）

　　　　　　　long 　64位（负2的63次方~2的63次方-1）

　　2.浮点数类型 float 　　32位 （3.4e-45~1.4e38）直接赋值时必须在数字后加上f或F

　　　　　　　　double　 64位 （4.9e-324~1.8e308）赋值时可以加d或D也可以不加
