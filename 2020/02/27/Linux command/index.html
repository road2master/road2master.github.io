五、平滑升级与回滚Nginx

平滑升级

  1. /home目录下解压新版本的Nginx

  2. 进入解压后的文件夹，配置新版本Nginx的安装目录

  3. 安装新版本Nginx

  4. 进入旧版本Nginx安装目录下的sbin目录，备份（重命名）旧版本Nginx的可执行文件

  5. 复制 新版本的可执行文件 到 旧版本的可执行文件的目录下（无需重命名）

  6. 平滑升级 并关闭旧版本的work进程（master进程暂不关闭，待服务正常运行后，可通过-QUIT信号指令关闭master进程）
